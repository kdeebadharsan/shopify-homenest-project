# HomeNest India - Shopify Store

This project is a Shopify development store created as part of an assignment.

## Features:
- Dawn theme customization
- Hero banner updated
- Custom button color
- 3 products added
- Bestsellers collection created
- Navigation menu setup
- Custom Liquid code (announcement bar)

## Announcement Bar:
Free Shipping on Orders Above Rs. 499
